# Tableau-sales-analytics-project
This project presents an interactive Sales Performance and Forecasting Dashboard built using Tableau. The dashboard analyzes historical sales data and provides insights into trends, discount impact, shipping performance, and future sales forecasting.
Project Overview
This project presents an interactive Sales Performance and Forecasting Dashboard built using Tableau.
The dashboard analyzes historical sales data and provides insights into trends, discount impact, shipping performance, and future sales forecasting.
